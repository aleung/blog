Sun JVM 1.3 жЬЙдЄ§зІНжЬАеЯЇжЬђзЪДеЖЕе≠ШжФґйЫЖжЦєеЉПпЉЪдЄАзІНзІ∞дЄЇcopyingжИЦscavengeпЉМе∞ЖжЙАжЬЙдїНзДґзФЯе≠ШзЪДеѓєи±°жРђеИ∞еП¶е§ЦдЄАеЭЧеЖЕе≠ШеРОпЉМжХіеЭЧеЖЕе≠Ше∞±еПѓеЫЮжФґгАВињЩзІНжЦєж≥ХжЬЙжХИзОЗпЉМдљЖйЬАи¶БжЬЙдЄАеЃЪзЪДз©ЇйЧ≤еЖЕе≠ШпЉМжЛЈиіЭдєЯжЬЙеЉАйФАгАВињЩзІНжЦєж≥ХзФ®дЇОminor collectionгАВеП¶е§ЦдЄАзІНзІ∞дЄЇmark-compactпЉМе∞ЖжіїзЭАзЪДеѓєи±°ж†ЗиЃ∞еЗЇжЭ•пЉМзДґеРОжРђињБеИ∞дЄАиµЈињЮжИРе§ІеЭЧзЪДеЖЕе≠ШпЉМеЕґдїЦеЖЕе≠Ше∞±еПѓдї•еЫЮжФґдЇЖгАВињЩзІНжЦєж≥ХдЄНйЬАи¶БеН†зФ®йҐЭе§ЦзЪДз©ЇйЧіпЉМдљЖйАЯеЇ¶зЫЄеѓєжЕҐдЄАдЇЫгАВињЩзІНжЦєж≥ХзФ®дЇОmajor collection.

еЬ®JVM 1.3еПКдї•еРОзЪДзЙИжЬђдЄ≠пЉМињШжЬЙеЕґдїЦеПѓйАЙзЪДеЖЕе≠ШжФґйЫЖжЦєж≥ХпЉМйАЪињЗзЙєеЃЪзЪДеПВжХ∞жЭ•иЃЊеЃЪгАВдЊЛе¶ВпЉЪеҐЮйЗПеЉПеЫЮжФґпЉМжѓПжђ°еП™е§ДзРЖдЄАе∞ПйГ®еИЖпЉЫжЫњдї£еНХзЇњз®ЛcopyingзЪДе§ЪзЇњз®Леєґи°МеЫЮжФґпЉЫжЫњдї£mark-compactзЪДconcurrent mark-sweepеЫЮжФґз≠Йз≠ЙгАВеПВиАГиµДжЦЩ[4][5]дЄ≠жЬЙжЫіе§ЪжППињ∞гАВ

JVMзЃ°зРЖзЪДеЖЕе≠ШпЉМйАЪеЄЄеПЂеБЪе†Ж(heap)пЉМеПѓдї•зФ®дЄЛйЭҐзЪДеЫЊжЭ•жППињ∞гАВ

JVMеРѓеК®еРОпЉМдњЭзХЩдЄАжЃµеЬ∞еЭАз©ЇйЧіпЉМињЩдЄ™з©ЇйЧізЪДе§Іе∞ПзФ±-XmxжМЗеЃЪгАВињЩеЭЧз©ЇйЧізЪДе§Іе∞Пе∞±жШѓheapеПѓиГљзЪДжЬАе§ІеАЉпЉМдљЖдЄАеЉАеІЛдЄНдЄАеЃЪеЕ®йГљеИЖйЕНдЇЖзЙ©зРЖеЖЕе≠ШпЉМеИЭеІЛеИЖйЕНзЪДheapе§Іе∞ПзФ±-XmsжМЗеЃЪпЉМе¶ВжЮЬ-Xmsе∞ПдЇО-XmxпЉМеЙ©дљЩйГ®еИЖжШѓvirtualзЪДпЉМељУйЬАи¶БзЪДжЧґеАЩпЉМеЖНеРСOSзФ≥иѓЈгАВ

-Xmx
set maximum Java heap size

-Xms
set initial Java heap size

-XX:MinHeapFreeRatio=40
Minimum percentage of heap free after GC to avoid expansion.

-XX:MaxHeapFreeRatio=70
Maximum percentage of heap free after GC to avoid shrinking.

-XX:NewRatio=2
Ratio of new/old generation sizes. [Sparc -client:8; x86 -server:8; x86 -client:12.]-client:8 (1.3.1+), x86:12]

-XX:NewSize=2.125m
Default size of new generation (in bytes) [5.0 and newer: 64 bit VMs are scaled 30% larger; x86:1m; x86, 5.0 and older: 640k]

-XX:MaxNewSize=
Maximum size of new generation (in bytes). Since 1.4, MaxNewSize is computed as a function of NewRatio.

-XX:SurvivorRatio=25
Ratio of eden/survivor space size [Solaris amd64: 6; Sparc in 1.3.1: 25; other Solaris platforms in 5.0 and earlier: 32]

-XX:PermSize=
Initial size of permanent generation

-XX:MaxPermSize=64m
Size of the Permanent Generation. [5.0 and newer: 64 bit VMs are scaled 30% larger; 1.4 amd64: 96m; 1.3.1 -client: 32m.]
