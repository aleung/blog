Javaзҡ„еҮ з§ҚXMLиҜӯжі•еҲҶжһҗAPI

еҸ‘иЎЁдәҺ 2003-09-08   |  

з”ЁJavaеҜ№XMLиҝӣиЎҢеӨ„зҗҶ, жңүеҘҪеӨҡз§Қж–№жЎҲ, еҰӮдёҚдәҶи§Јзӣёдә’зҡ„е…ізі»и®©дәәеҫҲеӨҙжҷ•. иҝҷйҮҢд»Ӣз»Қзҡ„йғҪжҳҜXML processing/parsingзҡ„иҢғз•ҙ, XMLзҡ„еә”з”Ёиҝҳжңүе…¶д»–зҡ„и®ёеӨҡж–№йқў, д№ҹжңүеҘҪеӨҡд»ӨдәәзңјиҠұзјӯд№ұзҡ„зј©еҶҷ.

XMLиҜӯжі•еҲҶжһҗзҡ„ж–№жЎҲжңүдёӨз§Қ, еҲҶеҲ«дёәDOMе’ҢSAX.

DOM: Document Object Model (http://www.w3.org/DOM/)
DOM з”ұ W3C еҲӣе»әпјҢе№¶дё”жҳҜиҜҘеҚҸдјҡзҡ„жӯЈејҸе»әи®®.
и§ЈжһҗеҷЁиҜ»е…Ҙж•ҙдёӘж–ҮжЎЈпјҢз„¶еҗҺжһ„е»әдёҖдёӘй©»з•ҷеҶ…еӯҳзҡ„ж ‘з»“жһ„пјҢз„¶еҗҺжӮЁзҡ„д»Јз Ғе°ұеҸҜд»ҘдҪҝз”Ё DOM жҺҘеҸЈжқҘж“ҚдҪңиҝҷдёӘж ‘з»“жһ„гҖӮжӮЁеҸҜд»ҘйҒҚеҺҶж ‘д»ҘдәҶи§ЈеҺҹе§Ӣж–ҮжЎЈеҢ…еҗ«дәҶд»Җд№ҲпјҢжӮЁеҸҜд»ҘеҲ йҷӨж ‘зҡ„еҮ дёӘйғЁеҲҶпјҢиҝҳеҸҜд»ҘйҮҚж–°жҺ’еҲ—ж ‘е’Ңж·»еҠ ж–°зҡ„еҲҶж”ҜпјҢзӯүзӯүгҖӮ
DOM жһ„е»әж•ҙдёӘж–ҮжЎЈй©»з•ҷеҶ…еӯҳзҡ„ж ‘гҖӮеҰӮжһңж–ҮжЎЈеҫҲеӨ§пјҢе°ұдјҡиҰҒжұӮжңүжһҒеӨ§зҡ„еҶ…еӯҳгҖӮ
DOM еҲӣе»әиЎЁзӨәеҺҹе§Ӣж–ҮжЎЈдёӯжҜҸдёӘдёңиҘҝзҡ„еҜ№иұЎпјҢеҢ…жӢ¬е…ғзҙ гҖҒж–Үжң¬гҖҒеұһжҖ§е’Ңз©әж јгҖӮеҰӮжһңжӮЁеҸӘйңҖе…іжіЁеҺҹе§Ӣж–ҮжЎЈзҡ„дёҖе°ҸйғЁеҲҶпјҢйӮЈд№ҲеҲӣе»әйӮЈдәӣж°ёиҝңдёҚиў«дҪҝз”Ёзҡ„еҜ№иұЎжҳҜжһҒе…¶жөӘиҙ№зҡ„гҖӮ
DOM и§ЈжһҗеҷЁеҝ…йЎ»еңЁжӮЁзҡ„д»Јз ҒеҸ–еҫ—жҺ§еҲ¶жқғд№ӢеүҚиҜ»еҸ–ж•ҙдёӘж–ҮжЎЈгҖӮеҜ№дәҺйқһеёёеӨ§зҡ„ж–ҮжЎЈпјҢиҝҷдјҡеј•иө·жҳҫи‘—зҡ„е»¶иҝҹгҖӮ

SAX API for XML (http://www.saxproject.org/)
SAX йҮҮз”ЁдәӢд»¶й©ұеҠЁзҡ„жЁЎејҸ, и§ЈйҮҠеҷЁиҜ»е…ҘXMLж–ҮжЎЈ, иҝӣиЎҢжү«жҸҸ, еҪ“еҸ‘зҺ°з¬ҰеҗҲжҢҮе®ҡжқЎд»¶зҡ„ж•°жҚ®ж—¶, еҗ‘жӮЁзҡ„д»Јз ҒеҸ‘йҖҒдёҖдёӘдәӢд»¶гҖӮSAX дёҚиғҪдҝ®ж”№ XML ж–ҮжЎЈ.
SAX дәӢд»¶жҳҜж— зҠ¶жҖҒзҡ„гҖӮеҪ“ SAX и§ЈжһҗеҷЁеңЁ XML ж–ҮжЎЈдёӯеҸ‘зҺ°ж–Үжң¬ж—¶пјҢе®ғе°ұеҗ‘жӮЁзҡ„д»Јз ҒеҸ‘йҖҒдёҖдёӘдәӢд»¶гҖӮиҜҘдәӢд»¶д»…д»…з»ҷжӮЁеҸ‘зҺ°зҡ„ж–Үжң¬пјӣе®ғдёҚе‘ҠиҜүжӮЁд»Җд№Ҳе…ғзҙ еҢ…еҗ«йӮЈдёӘж–Үжң¬гҖӮеҰӮжһңжӮЁжғізҹҘйҒ“иҝҷдёҖзӮ№пјҢеҲҷеҝ…йЎ»иҮӘе·ұзј–еҶҷзҠ¶жҖҒз®ЎзҗҶд»Јз ҒгҖӮ
SAX дәӢд»¶дёҚжҳҜжҢҒд№…зҡ„гҖӮеҰӮжһңеә”з”ЁзЁӢеәҸйңҖиҰҒдёҖдёӘж•°жҚ®з»“жһ„жқҘеҜ№ XML ж–ҮжЎЈе»әжЁЎпјҢеҲҷеҝ…йЎ»иҮӘе·ұзј–еҶҷйӮЈж ·зҡ„д»Јз ҒгҖӮеҰӮжһңжӮЁйңҖиҰҒд»Һ SAX дәӢд»¶и®ҝй—®ж•°жҚ®пјҢе№¶дё”жІЎжңүжҠҠйӮЈдёӘж•°жҚ®еӯҳеӮЁеңЁд»Јз ҒдёӯпјҢйӮЈд№ҲжӮЁдёҚеҫ—дёҚеҶҚж¬Ўи§ЈжһҗиҜҘж–ҮжЎЈгҖӮ
SAX зҡ„йҖҹеәҰзӣёеҪ“еҝ«, иҖҢдё”иҠӮзңҒеҶ…еӯҳ, еӣ дёәе®ғдёҚдјҡдҝқеӯҳж•°жҚ®еңЁеҶ…еӯҳд№Ӣдёӯ.

DOM/SAX жҳҜдёӨз§ҚдёҚеҗҢзҡ„и§Јжһҗж–№жі•, еҗ„жңүдҫ§йҮҚзӮ№, з”ЁдәҺдёҚеҗҢз”ЁйҖ”. е®ғд»¬йғҪеҸҜд»Ҙз”ЁдёҚеҗҢзҡ„зј–зЁӢиҜӯиЁҖжқҘе®һзҺ°, дёҚеҗҢзҡ„дҫӣеә”е•ҶдјҡжҸҗдҫӣеҗ„иҮӘзҡ„и§ЈйҮҠеҷЁзүҲжң¬. дҫӢеҰӮMicrosoftжҸҗдҫӣдәҶDOMеҲҶжһҗеҷЁmsxml.dll, Apacheзҡ„xerces, SUNзҡ„crimsonзӯү.

DOMдёҺSAX, еҰӮдҪ•йҖүжӢ©?

  • еә”з”ЁзЁӢеәҸе°ҶеҰӮдҪ•йғЁзҪІпјҹ
    еҰӮжһңжӮЁзҡ„еә”з”ЁзЁӢеәҸе°ҶиҰҒдҪңдёә Java applet йғЁзҪІпјҢйӮЈд№ҲжӮЁдјҡеёҢжңӣдҪҝиҰҒдёӢиҪҪзҡ„д»Јз Ғж•°йҮҸжңҖе°ҸпјҢSAX и§ЈжһҗеҷЁжҜ” DOM и§ЈжһҗеҷЁе°ҸгҖӮ

  • дёҖж—Ұи§ЈжһҗдәҶ XML ж–ҮжЎЈпјҢиҝҳйңҖиҰҒеӨҡж¬Ўи®ҝй—®йӮЈдәӣж•°жҚ®еҗ—пјҹ
    еҰӮжһңжӮЁйңҖиҰҒеӣһиҝҮеӨҙжқҘи®ҝй—® XML ж–Үд»¶зҡ„е·Іи§ЈжһҗзүҲжң¬пјҢDOM еҸҜиғҪжҳҜжӯЈзЎ®зҡ„йҖүжӢ©гҖӮиҖҢ SAX дәӢд»¶иў«и§ҰеҸ‘ж—¶пјҢеҰӮжһңжӮЁд»ҘеҗҺйңҖиҰҒе®ғпјҢеҲҷз”ұжӮЁпјҲејҖеҸ‘дәәе‘ҳпјүиҮӘе·ұеҶіе®ҡд»Ҙжҹҗз§Қж–№ејҸдҝқеӯҳе®ғгҖӮеҰӮжһңжӮЁйңҖиҰҒи®ҝй—®дёҚжӣҫдҝқеӯҳзҡ„дәӢд»¶пјҢеҲҷеҝ…йЎ»еҶҚж¬Ўи§ЈжһҗиҜҘж–Үд»¶гҖӮиҖҢ DOM иҮӘеҠЁдҝқеӯҳжүҖжңүзҡ„ж•°жҚ®гҖӮ

  • еҸӘйңҖиҰҒ XML жәҗж–Үд»¶зҡ„е°‘йҮҸеҶ…е®№еҗ—пјҹ
    еҰӮжһңжӮЁеҸӘйңҖиҰҒ XML жәҗж–Үд»¶зҡ„е°‘йҮҸеҶ…е®№пјҢйӮЈд№Ҳ SAX еҸҜиғҪжҳҜжӯЈзЎ®зҡ„йҖүжӢ©гҖӮSAX дёҚдјҡдёәжәҗж–Үд»¶дёӯзҡ„жҜҸдёӘдёңиҘҝеҲӣе»әеҜ№иұЎпјӣжӮЁиҰҒзЎ®е®ҡд»Җд№ҲжҳҜйҮҚиҰҒзҡ„гҖӮдҪҝз”Ё SAXпјҢжӮЁиҰҒжЈҖжҹҘжҜҸдёӘдәӢд»¶д»ҘдәҶи§Је®ғжҳҜеҗҰдёҺжӮЁзҡ„йңҖиҰҒжңүе…іпјҢз„¶еҗҺзӣёеә”ең°еӨ„зҗҶе®ғгҖӮжӣҙеҰҷзҡ„жҳҜпјҢдёҖж—ҰжүҫеҲ°жӮЁжӯЈеңЁеҜ»жүҫзҡ„дёңиҘҝпјҢжӮЁзҡ„д»Јз Ғе°ұдјҡжҠӣеҮәдёҖдёӘејӮеёёжқҘе®Ңе…ЁеҒңжӯў SAX и§ЈжһҗеҷЁгҖӮ

  • жӮЁжӯЈеңЁдёҖеҸ°еҶ…еӯҳеҫҲе°‘зҡ„жңәеҷЁдёҠе·ҘдҪңеҗ—пјҹ
    иӢҘжҳҜзҡ„иҜқпјҢдёҚз®ЎжӮЁеҸҜиғҪиҖғиҷ‘еҲ°зҡ„е…¶е®ғеӣ зҙ жҳҜд»Җд№ҲпјҢSAX жҳҜжӮЁзҡ„жңҖдҪійҖүжӢ©гҖӮ

еңЁJavaиҜӯиЁҖдёӯ, иҰҒеӨ„зҗҶXML, йҷӨдәҶзӣҙжҺҘдҪҝз”ЁDOM/SAX, иҝҳжңүй«ҳеұӮж¬ЎдёҖзӮ№зҡ„APIеҸҜд»ҘдҪҝз”Ё, жҜ”иҫғеёёи§Ғзҡ„жҳҜJAXPе’ҢJDOM.

JAXP: Java API for XML Processing
жҳҜ Sun жҸҗдҫӣзҡ„ж ҮеҮҶAPI, еҢ…еҗ«еңЁJDKдёӯ.
дҪҶе®ғжӣҙеӨҡзҡ„жҳҜдёҖеҘ—жҺҘеҸЈ, е°ҶDOMе’ҢSAXзҡ„жҺҘеҸЈи§„иҢғиө·жқҘ, е№¶жІЎжңүеңЁжҳ“ж“ҚдҪңжҖ§ж–№йқўжҸҗдҫӣеӨӘеӨҡзҡ„ж–№дҫҝ.

JDOM: з”ұBrett McLaughlin, Jason HunterдёӨдәәеҲӣз«Ӣзҡ„open sourceйЎ№зӣ® (http://www.jdom.org)
JDOMзҡ„и®ҫи®Ўзӣ®зҡ„дёҚжҳҜиҰҒеҒҡдёҮиғҪзҡ„и§ЈжһҗеҷЁ, иҖҢжҳҜиғҪжҸҗдҫӣеёёз”Ёзҡ„еӨ§йғЁеҲҶзҡ„еҠҹиғҪ. е®ғзҡ„дҪҝз”ЁзӣёеҜ№з®ҖеҚ•, е®№жҳ“зҗҶи§Ј, дҪҝз”Ёе®ғејҖеҸ‘д»Јз ҒйҮҸдјҡжҜ”иҫғе°‘.
JDOM зҡ„йҖҹеәҰд№ҹжҜ”иҫғй«ҳ, д»…д»…жҜ”SAXж…ўдёҖзӮ№. е®ғдё»иҰҒжҳҜдҪҝз”ЁSAXжқҘе®һзҺ°еҠҹиғҪ, еҪ“йңҖиҰҒз”ЁеҲ°DOMзҡ„ж—¶еҖҷжүҚдҪҝз”ЁDOM.

иҝҳжңүдёҚе°‘зұ»дјјзҡ„API, жҹҗдәӣз”ҡиҮіе·Із»ҸжҜ”JDOMжӣҙеҘҪ, дёҚиҝҮйғҪдёҚеҰӮJDOMдҪҝз”ЁиҢғеӣҙе№ҝ.

JAXP/JDOM йғҪжҳҜе»әз«ӢеңЁ DOM/SAX д№ӢдёҠзҡ„API, д№ҹеҸҜи®ӨдёәжҳҜжҠҪиұЎеұӮ, йңҖиҰҒйҖҡиҝҮе…·дҪ“дҫӣеә”е•ҶжҸҗдҫӣ DOM/SAX е®һзҺ°жқҘе®ҢжҲҗ XML и§Јжһҗе·ҘдҪң. дҪңдёәдёҖдёӘзӣёеҜ№й«ҳеұӮзҡ„жҺҘеҸЈ, йҡҗи—ҸдәҶе…·дҪ“еҺӮе•Ҷе®һзҺ°, еҮҸдҪҺеӨҚжқӮжҖ§, йҷҚдҪҺејҖеҸ‘зҡ„е·ҘдҪңйҮҸ.

дёҖиҲ¬жқҘиҜҙ, еңЁжҲ‘д»¬ејҖеҸ‘зЁӢеәҸзҡ„ж—¶еҖҷ, д»ҺејҖеҸ‘ж•ҲзҺҮ,е…је®№жҖ§,з§»жӨҚжҖ§зӯүж–№йқўиҖғиҷ‘, йғҪжҳҜйҖүжӢ©дҪҝз”Ёй«ҳеұӮзҡ„APIжӣҙеҘҪдәӣ. JAXPе·Із»Ҹйҷ„еёҰеңЁJDKдёӯ, з”ұSUNжүҖж”ҜжҢҒ, еә”иҜҘж ҮеҮҶеҢ–/е»¶з»ӯжҖ§ж–№йқўдјҡжҜ”иҫғеҘҪ; иҖҢж №жҚ®зҪ‘дёҠејҖеҸ‘иҖ…еҸҚжҳ , жҷ®йҒҚйғҪе–ңж¬ўз”ЁJDOM, еӣ дёәдҪҝз”Ёз®ҖеҚ•ж–№дҫҝ.
